# Lejun's Scoop Bucket

[![Tests](https://github.com/aik2mlj/scoop-bucket/actions/workflows/ci.yml/badge.svg)](https://github.com/aik2mlj/scoop-bucket/actions/workflows/ci.yml) [![Excavator](https://github.com/aik2mlj/scoop-bucket/actions/workflows/excavator.yml/badge.svg)](https://github.com/aik2mlj/scoop-bucket/actions/workflows/excavator.yml)

This is a bucket for [Scoop](https://scoop.sh), the Windows command-line installer.

Maintained software:

- [canvas-downloader](https://github.com/aik2mlj/canvas-downloader)

## How do I install these software?

```pwsh
scoop bucket add aik2mlj https://github.com/aik2mlj/scoop-bucket
scoop install aik2mlj/<software>
```
